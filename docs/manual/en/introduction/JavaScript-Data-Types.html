Understanding Javascript Data types is not a prerequisite, but It would be helpful if you do.

  • Number: numbers in JavaScript can be integers or floats (have decimals). While in Python an integer is a specific number type and a float is another specific number type.
    • // JavaScript let x = 6; // number let y = 6.1; // number # Python x = 6 # integer y = 6.1 # float
  • Boolean: boolean in JavaScript is in lower case while boolean in Python is capitalized
    • // JavaScript let isTrue = true; let isFalse = false; # Python isTrue = True isFalse = False
  • Strings: Strings in JavaScript is same as Strings in Python. In both languages string are identified by quotes(single or double)
    • // JavaScript let message = 'Hello World!'; // can be either single or double quotes # Python message = 'Hello World!' # can be either single or double quotes
  • Array (or List in Python): Array in JavaScript is the same as List in Python. Both can store a list of various data types.
    • // JavaScript let myList = [ 5.2, 'Orange', true ]; # Python myList = [ 5.2, 'Orange', True ]

  • Undefined: Also, there's not `undefined` keyword in python, but undefined indicates that a variable has been declared but not given a value.
  • Objects: JavaScript objects are like Python classes (because they inherit like Python classes)
