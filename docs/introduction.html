Introduction

PyWeb3D is three.js with python syntax.

The goal of PyWeb3D is to enable python developers to tap into the beautiful world of 3D on the web using a language that feels natural.

lets briefly go over the tools that made PyWeb3D possible.

  • Brython
  • Three.js

Brython

Brython is a Python 3 implementation for client-side web programming, designed to replace Javascript as the scripting language for the Web.
visit brython's website to learn more..

Three.js

Three.js, world's most popular JavaScript framework for displaying 3D content on the web. It aim to create an easy to use, lightweight, cross-browser, general purpose 3D library.
Visit three.js website to learn more about the library.

This documentation will aid you to merge these two powerful libraries.
