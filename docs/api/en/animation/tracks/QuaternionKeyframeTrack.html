[name]

A Track of quaternion keyframe values.

Constructor

[name]( [param:str name], [param:list times], [param:list values] )

[page:str name] (required) identifier for the KeyframeTrack.
[page:list times] (required) list of keyframe times.
[page:list values] values for the keyframes at the times specified, a flat list of quaternion components.
[page:Constant interpolation] the type of interpolation to use. See [page:Animation Animation Constants] for possible values. Default is [page:Animation InterpolateLinear].

Instance Attributes

See [page:KeyframeTrack] for inherited attributes.

[property:Constant DefaultInterpolation]

The default interpolation type to use, [page:Animation InterpolateLinear].

[property:str ValueTypeName]

Returns value type name. Example: 'quaternion'.
