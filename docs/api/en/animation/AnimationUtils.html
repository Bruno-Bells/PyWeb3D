[name]

An object with various functions to assist with animations, used internally.

Methods

[method:list arraySlice]( list, from, to )

This method works the same way as Python `slice`, or JavaScript `Array.prototype.slice` but also works on typed arrays in JavaScript. Examples of JavaScript typed array: `Int8Array`, `Uint8Array`, `Uint8ClampedArray` and more.

[method:list convertArray]( list, type, forceClone )

Converts a list to a specific type in JavaScript.

[method:list flattenJSON]( jsonKeys, times, values, valuePropertyName )

Used for parsing AOS keyframe formats.

[method:list getKeyframeOrder]( times )

Returns a list by which times and values can be sorted.

[method:bool isTypedArray]( object )

Returns `True` if the object is a JavaScript typed array.

[method:AnimationClip makeClipAdditive]( [param:AnimationClip targetClip], [param:int referenceFrame], [param:AnimationClip referenceClip], [param:int fps] )

Converts the keyframes of the given animation clip to an additive format.

[method:list sortedArray]( values, stride, order )

Sorts the array previously returned by [page:AnimationUtils.getKeyframeOrder getKeyframeOrder].

[method:AnimationClip subclip]( [param:AnimationClip clip], [param:str name], [param:int startFrame], [param:int endFrame], [param:int fps] )

Creates a new clip, containing only the segment of the original clip between the given frames.
