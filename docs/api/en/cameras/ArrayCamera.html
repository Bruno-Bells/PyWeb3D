[name]

[name] can be used in order to efficiently render a scene with a predefined set of cameras. This is an important performance aspect for rendering VR scenes.
An instance of [name] always has a list of sub cameras. It's mandatory to define for each sub camera the `viewport` attribute which determines the part of the viewport that is rendered with this camera.
