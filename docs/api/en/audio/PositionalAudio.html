[name]

Create a positional audio object.

This uses the [link:https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API Web Audio API].

Code Example

# create an AudioListener and add it to the camera listener = AudioListener() camera.add( listener ) # create the PositionalAudio object (passing in the listener) sound = PositionalAudio( listener ) # load a sound and set it as the PositionalAudio object's buffer audioLoader = AudioLoader() def audio_settings(buffer): sound.setBuffer( buffer ) sound.setRefDistance( 20 ) sound.play() audioLoader.load( 'sounds/song.ogg', audio_settings) # create an object for the sound to play from sphere = SphereGeometry( 20, 32, 16 ) material = MeshPhongMaterial( { 'color': 0xff2200 } ) mesh = Mesh( sphere, material ) scene.add( mesh ) # finally add the sound to the mesh mesh.add( sound )
