[name]

The [name] represents a virtual [link:https://developer.mozilla.org/en-US/docs/Web/API/AudioListener listener] of all the positional and non-positional audio effects in the scene.
A PyWeb3D application usually creates a single instance of [name]. It is a mandatory construtor parameter for audios entities like [page:Audio Audio] and [page:PositionalAudio PositionalAudio].
In most cases, the listener object is a child of the camera. So the 3D transformation of the camera represents the 3D transformation of the listener.

Code Example

# create an AudioListener and add it to the camera listener = AudioListener() camera.add( listener ) # create a global audio source const sound = Audio( listener ) # load a sound and set it as the Audio object's buffer audioLoader = AudioLoader() def audio_settings(buffer): sound.setBuffer( buffer ) sound.setLoop( True ) sound.setVolume( 0.5 ) sound.play() audioLoader.load( 'sounds/ambient.ogg', audio_settings)
